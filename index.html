Featured Project

FitCheck

FitCheck is a product concept designed to reduce clothing returns caused by incorrect size selection in ecommerce.

Problem

Fashion ecommerce stores lose revenue and customer satisfaction due to returns caused by incorrect sizing decisions.

Solution

FitCheck provides personalized size recommendations that help shoppers choose the correct size before completing a purchase.
